什么是Github Pages？

GitHub Pages 是 GitHub 提供的一个免费的静态网站托管服务，它允许 GitHub 用户创建和托管自己的静态网站，这些网站可以通过特定的 GitHub 仓库进行管理和托管。用户只需在自己的 GitHub 帐户中创建一个特定名称的仓库，将网站文件上传到该仓库即可

什么是网站文件？

网站文件包含网站的本文、图像、样式表、脚本、配置文件等。网站文件通常包含以下内容：

HTML文件、CSS文件、JavaScript文件、图像文件、字体文件、配置文件、服务器脚本文件、样本文件、其他资源文件等等。

一个最简单的网站文件包含：

HTML、CSS、JavaScript、图像

Hexo和Nginx的区别是什么？

  1. Hexo：是一个静态网站生成器，用户编写markdown文章，hexo将这些文章转换为静态HTML网页。
  2. Nginx：是一个Web服务器，旨在处理网络请求

关联：

​ Hexo生成网站文件，Nginx负责对这些文件进行托管和服务。
